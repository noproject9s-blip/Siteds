# Sellix Telegram Bot (RU)

Готовый шаблон Telegram-бота для продажи цифровых товаров:
- файлы и фото;
- игровые аккаунты;
- донаты и услуги;
- внутренняя цифровая валюта.

## Концепт валюты
Валюта бота: **NOVA**.

- Курс: **1 NOVA = 15 ₽**.
- NOVA используется для покупки товаров.
- Администратор может пополнять NOVA пользователям.

## Быстрый старт

1. Установите Python 3.11+
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Скопируйте `.env.example` в `.env` и заполните:

```bash
cp .env.example .env
```

4. Запустите:

```bash
python -m bot.main
```

## Основные команды
- `/start` — запуск бота
- `/catalog` — каталог товаров
- `/balance` — баланс в NOVA
- `/buy <id>` — купить товар
- `/topup <amount>` — рассчитать пополнение в рублях

## Админ-команды
- `/admin_add_item` — добавить товар
- `/admin_stock` — остатки
- `/admin_add_nova <user_id> <amount>` — начислить NOVA

## Файлы дизайна
Готовые материалы лежат в `assets/`:
- `avatar.svg` — аватар бота
- `banner.svg` — баннер для канала/поста
- `story-banner.svg` — вертикальный баннер

Можно экспортировать в PNG через любой редактор (Figma, Inkscape, Photopea).
